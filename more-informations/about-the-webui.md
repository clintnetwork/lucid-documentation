# About the WebUI

## Introduction

Lucid will embed a beautiful dashboard created in [VueJS](http://vuejs.org) to easily manage your node, by example:

* Handle Stored Data
* Issue Authentication Tokens
* Manage Distributed Instances
* Manage ACLs or Encryption
